# Backend

## Description

This is the backend for the Lancerr application.

## Dependencies

- Express.js
- MongoDB
- Bcrypt
- Cookie-parser
- CORS
- Cryptr
- Socket.io

## Scripts

- `start`: Starts the server using `node server.js`.
- `dev`: Starts the server in development mode using nodemon.
- `test`: Placeholder script for running tests.
